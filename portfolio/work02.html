scattered love

coding project made in p5.js web editor, implementing handfrees.js

Needs use of camera. Takes a while to load. Can lag depending on screen size.

Code that explores personal grief. played with the concept of grasping a "scattering" memory:

the harder you try to remember it, the faster it fades.

Project was coded in p5.js; it uses the handfree.js library. Detects when a person does a "hand-pinch" to transition between images.

Images used in project that were modified with p5 pixel.

Design came about through multiple iterations. Needed something that would work with code well. More abstract than detailed.
