tiny worlds

collaborator: maria paz martinez

augmented reality project made with adobe after effects and adobe aero (beta)

File size of this project is large. May take a while to load.

To experience this project, please scan QR code with iOS operated phone or iPad.

This project was created to model tiny planets like in the little prince. Our goal was to make a cute and immersive environment that used elements like animation and audio.

The behaviors of interactivity include: proximity enter, which starts the animation and triggers audio, and tap, which makes the chosen world orbit around the camera.

My responsibility in the project was in creating and exporting assets and in animating both the "ocean" world and "space" world.

Although aero was simple and easy to use, there were limitations in how interactive our experience could be because of what functions are available with aero currently in beta. Would have liked to use a more customizable program next time.

Each world was created in after effects. It was inspired by this base "space world" animation. All animations ended up being 2D because of how we imported our files in aero: we needed to import them as a png image sequence.
